# Customer Journey Map
### StockPulse — Trading Platform Simulation
**Phase 2: Requirement Analysis**

---

## Overview

This customer journey map documents the complete experience of a new user (beginner investor / student) from first discovering StockPulse to becoming a confident regular user.

**User Persona:** Riya, 21, CS student, no prior investing experience.
**Goal:** Learn stock trading safely and track portfolio performance.

---

## Journey Stages

### Stage 1: AWARENESS
**How does the user find StockPulse?**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| GitHub / College submission | Sees the project linked by professor | Curious |
| Vercel live URL | Opens the website for the first time | Interested |
| Landing page | Reads the hero section and feature highlights | Impressed |

**User Thoughts:**
> "This looks clean and professional. It says I get $100,000 to start — let me try this."

**Pain Points:**
- Might be skeptical — "Is this actually real-time?"

**Opportunities:**
- Strong landing page with clear value proposition
- Showcase section showing live stock prices immediately

---

### Stage 2: ONBOARDING
**User registers and explores the platform**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| Auth page | Clicks Register, fills in name/email/password | Neutral |
| Registration success | Gets redirected to Markets page with $100,000 balance | Excited |
| Navigation bar | Sees balance displayed prominently | Motivated |
| Guide page | Reads the step-by-step "How to Trade" guide | Informed |

**User Thoughts:**
> "Wow, I actually have $100,000! The guide makes it really clear what to do next."

**Pain Points:**
- May feel overwhelmed by 50+ stocks initially

**Opportunities:**
- Guide page (01–04 steps) explains everything clearly
- Featured stocks on homepage showcase to highlight top picks

---

### Stage 3: EXPLORATION
**User browses the markets**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| Markets page | Browses stocks in grid or list view | Curious |
| Search bar | Searches for a known company (e.g., Apple) | Comfortable |
| Sort buttons | Sorts by price or name | In control |
| Live price badges | Watches prices update in real time | Amazed |
| Stock card | Clicks on a stock to view details | Engaged |

**User Thoughts:**
> "The prices are actually moving! This feels like a real trading app."

**Pain Points:**
- Wants to know which stocks are "good" to buy
- May not understand what exchange/industry means

**Opportunities:**
- Stock detail page with description, IPO date, and site link
- Industry tags for context

---

### Stage 4: FIRST TRANSACTION
**User buys their first stock**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| Stock detail page | Reviews stock info and current price | Thinking |
| Transaction form | Enters number of shares to buy | Nervous |
| Buy confirmation | Sees balance deduct and portfolio update | Thrilled |
| Portfolio page | Sees first holding appear | Proud |

**User Thoughts:**
> "I just bought my first stock! My balance went down but now I own shares."

**Pain Points:**
- Uncertainty about how many shares to buy

**Opportunities:**
- Transaction form shows total cost preview
- Instant balance update builds trust

---

### Stage 5: PORTFOLIO MANAGEMENT
**User tracks and manages investments**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| Investments page | Views purchased stocks with current value | Analytical |
| Profit/loss badges | Sees green/red indicators for each stock | Emotionally invested |
| Sell action | Decides to sell a stock for profit | Satisfied |
| Transaction history | Reviews all past trades in Dashboard | Reflective |
| Activity logs | Reviews account activity | Organised |

**User Thoughts:**
> "I made a profit on Apple! The transaction history helps me understand my decisions."

**Pain Points:**
- Wants more insight into why prices moved

**Opportunities:**
- Insights chart showing portfolio balance over time
- P&L calculation per stock

---

### Stage 6: RETENTION
**User returns regularly**

| Touchpoint | User Action | Emotion |
|------------|-------------|---------|
| Return visit | Logs back in to check portfolio | Habitual |
| Dashboard | Updates display name in account settings | Personalised |
| Markets | Discovers new stocks to invest in | Curious again |

**User Thoughts:**
> "I check this every day now. It's helping me understand how the market works."

---
